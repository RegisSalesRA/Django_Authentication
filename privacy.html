Políticas de Privacidade

1. Privacidade dos dados coletado

Esta Política de Privacidade aplica-se a todos os usuários e visitantes do site e integra os Termos de Condições Gerais de Uso do site, doravante dominada a Oxydelta.
Práticas e processos adotadas por nossa empresa, para que seja transparente para nossos usuários. Aqui iremos informar sobre todos os direitos, garantias, formas de uso, dados recolhidos, processamento e descarte de informações pessoais, que o usuário/visitante tem.

1.1. Para que finalidade nos usaremos os seus dados?

Os dados pessoais do usuário e do visitante coletados e armazenados pela plataforma Oxydelta tem por finalidade:

  • Bem estar do usuário/visitante: aprimorar nossos serviços oferecidos, facilitar, agilizar e cumprir os compromissos estabelecidos entre o usuário e a empresa, melhora a experiência dos usuários e fornecer funcionalidades específicas a depender das características básicas do usuário.
  • Melhorias da plataforma: compreeender como o usuário utiliza os serviços da plataforma, par ajudar no desenvolvimento de negócios e técnicas.
  • Anúncios: apresentar anúncios personalizados para o usuário com base nos dados oferecidos.
  • Comerciais: os dados são usados para personalizar o conteúdo oferecido e gerar subsídio à plataforma para a melhora da qualidade no funcionamento dos serviços.
  • Previsão do perfil do usuário: tratamento automatixados de dados pessoais para avaliar o uso na plataforma
  • Previsão do perfil do usuário: tratamento automatizado de dados pessoais para avaliar o uso da plataforma.
  • Dados de cadastro: para permitir o acesso do usuário a determinados conteúdos da plataforma, exclusivo para usuários cadastrados.
  • Dados de contrato: conferir às partes da segurança jurídica e facilitar a conclusão de negócio.

    • O tratamento de dados pessoais para finalidades não previstas nesta Política de Privacidade somente ocorrerá mediante comunicação prévia ao usuário, de modo que os direitos e obrigações aqui previstos permanecem aplicáveis.


    1.2. Como recolhemos os dados pessoais do usuário e do visitante?

    Os dados pessoais são recolhidos pela plataforma através de formulários, ao criar uma conta/perfil na plataforma: email, nome completo, idade, cidade de residência e profissão.

    Quando um usuário/visitante acessa páginas do site: as informações sobre interação e a cesso são coletadas pela empresa para garantir uma melhor experiência ao usuário e visitante. Estes dados podem tratar sobre as palavras-chaves utilizadas em busca, o Compartilhamento de um documento específico, comentários, visualizações de páginas, URL de onde o usuário e visitante provêm, o navegador que utilizam e seus IPs de acessp, dentre outras que poderão ser armazenadas e retidas.

    A plataforma também recebe informaçõe de terceiros quando o usuario faz log in com conta do Google ou Facebook.A utilização desses dados é autorizada previamente pelos usuários junto ao terceiro.


    1.3. Quais dados pessoais recolhemos sobre o usuário e visitante?

    Os dados pessoais do usuário ou visitante recolhidos são:

  • Dados para criação de conta/perfil na plataforma: e-mail, nome completo, cidade de residência e profissão.
  • Dados para otimização da navegação: acesso a páginas, palavras-chaves utilizadas na busca, recomendações, comentários, interações, tais como, número do cartão de crédito e outras informações sobre o cartãi, além dos pagamentos efetuados.


    • 1.4. Compartilhamento de dados

    O compartilhamento de dados do usuário ocorre apenas com os dados referentes a publicações realizadas pelo prórpio usuário, tais ações são compartilhadas publicamente com os outros usuários.

    Os dados do perfil do usuário são compartilhados publicamente em sistemas de busca e dentro da plataforma, sendo permitido ao usuário modificar tal configuração para que seu perfil não apareça nos resultados de busca de tais ferramentas.


    1.5. Os dados pessoais armazenados serão transferidos a terceiros?

    Os dados pessoais (não) podem ser compartilhados (com terceiros).

    1.6. Por quanto tempo os dados serão armazenados?

    Os dados podem ser removidos ou anonimizados a pedido do usuário, executando os casos em que a lei oferecer outro tratamento.
    Ainda, os dados pessoais do usuário apenas podem ser conservados após o término de seu tratamento nas seguintes hipóteses prevista no art.16

  • Cumprimento de obrigação legal ou regulatória pelo controlador.
  • Estudo por órgão de pesquisa, garantida, sempre que possível, a anonimização dos dados pessoais.
  • Tranferência a terceiro, desde que respeitados os requisitos de tratamento de dados dispostos nesta Lei.
  • Uso exclusivo do controlador, vedado seu acesso por terceiro, e desde que anonimizados os dados.


    • 1.7. Alterações para essa política de privacidade

    Reservamos o direito de modificar esta Política de Privacidade a qualquer momento, portanto, é recomendável que o usuário e visitante revise-a com frequência.
    As lterações e esclarecimentos vão surtir efeito imediatamente após sua publicação na plataforma. Quando realizadas alteraçõe os usuários serão notificados. Ao utilizar o serviço ou fornecer informações pessoais após eventuais modificações.


    2. Entenda melhor suas necessidades

    Os cookies refrem-se a arquivos de texto enviado pela plataforma ao computador do usuário e visitante e ue nele ficam aramzenados, com informações relacionadas á navegação do site. Tais informações são relacionadas aos dados d e acesso como local e horário de acesso e são armazenados pelo navegador do usuário e visitante para que o servidor da plataforma possa lê-las posteriormente a fim de personalizar os serviços da plataforma.

    O usuário e o visitante da plataforma Oxydelta manifesta conhecer e aceitar que pode ser utilizado um sistema de coleta de dados de navegação mediante á utilização de cookies.

    O cookie persistente permanece no disco rígido do usuário e visitante depois que o navegador é fechado e será usado pelo navegador em visitas subsequentes ao site. Os cookies persistentes podem ser removidos seguindo as instruções do seu navegado. Já o cookie de sessão é temporario e desparece depois que o navegador é fechado. É possível redefinir o seu navegador da web para recusar todos os cookies, porém alguns recursos da plataforma podem não funcionar corretamente se a capacidade de aceitar cookies estiver desabilitada.
